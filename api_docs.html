📚 API Documentation

Use the following endpoints to integrate TAKER SMS into your applications.

🔑 Base URL

https://taker-sms.onrender.com

🔐 Authentication

Include your API key in the header:

mauthapi: YOUR_API_KEY
POST /api/v1/get-number

Request new virtual numbers

{
    "prefix": "26134",
    "quantity": 1
}

Response:

{
    "success": true,
    "numbers": ["26134123456"]
}
GET /api/v1/check-otp

Get received OTP codes

Response:

{
    "otps": [{
        "number": "26134123456",
        "code": "490782",
        "message": "490782 is your Instagram code.",
        "app": "Instagram"
    }]
}
GET /api/v1/numbers

List all allocated numbers

GET /api/v1/balance

Check your balance

POST /api/v1/delete-number

Delete a specific number

{
    "number_id": 1
}
